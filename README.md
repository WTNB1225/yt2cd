# yt2cd
YouTubeから音楽をダウンロードし、オーディオCD作成用のISOイメージファイルを生成する自動化ツールです。

## 概要
- お気に入りのYouTubeプレイリストや動画から音源を抽出し、cdを焼くために必要なisoファイルを作成します。
- 動画タイトルが **`Artist - Title`（洋楽でよくある表記）** の形式の場合、  
  - `artist`
  - `title`  

  の **ID3メタデータを自動設定**します。


## 使い方
```bash
python main.py <YouTubeリンク> <出力ファイル名> <ボリューム名> [options]
```

| 引数 | 型 | 必須 | 説明 |
|-----|-----|-----|-----|
| url | str | Yes | ダウンロード対象のYouTube動画またはプレイリストのURLを指定します。 |
| iso_name | str | Yes | 生成するISOファイルの保存名（拡張子 `.iso` を含む）を指定します。 |
| volume | str | Yes | ISOファイルのボリュームラベル（ディスクを読み込んだ際に表示される名前）を指定します。 |
| -d, --outdir | str | No | MP3ファイルの一時保存先ディレクトリを指定します。指定しない場合は実行ディレクトリ（`./`）に保存されます。 |
| -h, --help | - | No | ヘルプメッセージを表示して終了します。 |
